# Кровля.Эксперт — лендинг

Статический сайт: HTML + CSS + JS, без сборки. Готов к деплою на GitHub Pages.

Все фотографии в WebP (общий вес папки images/ — ~3.7 МБ, LCP < 1.5 с).

---

## 🚀 Быстрый деплой на GitHub Pages (10 минут)

### Шаг 1. Создайте репозиторий
1. Заходите на github.com → **New repository**
2. Название: любое, например `krovlya-expert`
3. Тип: **Public** (для бесплатного GitHub Pages)
4. Не ставьте галочку «Add README» — у нас уже свой

### Шаг 2. Загрузите файлы
Через веб-интерфейс GitHub:
1. На странице пустого репозитория → **uploading an existing file**
2. Перетащите **всё содержимое папки** (не саму папку) — `index.html`, `styles.css`, `script.js`, папку `images`, все остальные файлы включая `.nojekyll`
3. Commit changes

Через терминал (если умеете):
```bash
git init
git add .
git commit -m "Initial release"
git branch -M main
git remote add origin https://github.com/USERNAME/REPO.git
git push -u origin main
```

### Шаг 3. Включите GitHub Pages
1. В репозитории → **Settings** → **Pages**
2. **Source**: Deploy from a branch
3. **Branch**: `main`, папка `/ (root)` → Save
4. Через 1–2 минуты сайт будет доступен по адресу:
   `https://USERNAME.github.io/REPO/`

### Шаг 4. (Опционально) Свой домен
1. Купите домен где угодно (reg.ru, GoDaddy…)
2. В настройках домена добавьте CNAME запись:
   `www` → `USERNAME.github.io`
3. В GitHub → Settings → Pages → Custom domain: `www.your-domain.ru`

Обязательно проверьте и замените в `index.html` и `script.js`:

### 1. Токен Telegram-бота ⚠️ КРИТИЧНО
Старый токен `8835407443:...` **скомпрометирован** (лежал в открытом доступе). Обязательно:
1. Откройте [@BotFather](https://t.me/BotFather) → `/revoke` → выберите бота → получите **новый** токен.

### 7. Домен (SEO) ⚠️ ВАЖНО
После деплоя замените плейсхолдер `YOUR-DOMAIN.github.io` на ваш реальный домен **во всех трёх файлах** (проще всего «Найти и заменить» по всей папке):

```bash
# Установите любой из вариантов
npx serve .
# ИЛИ
python3 -m http.server 8000
```

**Не открывайте `index.html` двойным кликом** — часть браузеров блокирует загрузку фото и `fetch` при протоколе `file://`.

## 🐛 Известные ограничения

1. **Токен бота публичен** — неизбежно для статических сайтов. Смягчено honeypot-полем и rate-limit'ом (1 заявка/30 сек). При злоупотреблении — перевыпуск токена.
2. **Форма «Ремонт» в мини-квизе** — использует фото `case-2-before.webp` (зимний дом со старой красной крышей). Если нужно уникальное фото, промпт есть в чате с ассистентом.
3. **Портреты Дмитрия и Игоря** — сгенерированные ИИ. При замене реальными фото сохраните пропорцию (примерно 3:2 landscape, лицо в верхней трети).

## 📊 Ожидаемые метрики

При правильной настройке рекламы (Директ / Google Ads):
- **CR на форму** (от посетителя до заявки в Telegram): 3–7%
- **Средняя цена лида** в тематике «монтаж кровли МО»: 500–1500 ₽
- **Bounce rate**: 30–45% (Hero + Боль №1 удерживают внимание)
- **LCP** (Largest Contentful Paint): < 1.5 с на 4G

Проверить эти метрики можно через:
- PageSpeed Insights: https://pagespeed.web.dev/
- WebPageTest: https://www.webpagetest.org/
